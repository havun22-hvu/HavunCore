TOTAL COMMANDER SFTP INSTALLATIE:
---------------------------------

1. Download SFTP plugin:
   https://www.ghisler.com/plugins.htm
   Zoek naar "SFTP" onder "File System Plugins"

2. Installeer plugin:
   - Download SFTP.zip
   - In TC: Configuration → Options → Plugins → WFX plugins
   - Add → selecteer sftp.wfx uit zip
   - Of: dubbelklik op zip in TC, hij vraagt automatisch om te installeren

3. Verbinding maken:
   - Ga naar: Network Neighborhood (of typ \\SFTP in adresbalk)
   - "Quick Connection" of "New Connection"

4. VPS connectie instellen:
   - Session name: Havun Server
   - Host: 188.245.159.115
   - Port: 22
   - Username: root
   - Authentication: Public key
   - Private key file: C:\Users\henkvu\.ssh\id_ed25519

   LET OP: Selecteer de PRIVATE key (ZONDER .pub extensie!)
